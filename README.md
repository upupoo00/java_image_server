# java_image_server
# 图书服务器项目
## 项目中用到的技术
- 前端：ajax jquery Vue
- 文件上传(使用io操作实现)
- 基于md5校验上传文件是否重复
- 基于Servlet返回图片二进制数据，供图片展示
- 基于请求头Referer及白名单列表实现图片防盗链
- 基于Filter实现用户会话统一管理

   Session的管理: 登录和敏感资源都有
   
   统一: 一处代码就实现
